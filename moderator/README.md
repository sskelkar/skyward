# Moderator Materials

This folder contains everything you need to facilitate a Skyward workshop.

## Files

### `GAMERUNNER.md`
**The complete facilitator guide** (28KB) - Everything you need to run the workshop:
- Pre-game preparation checklist
- Quarter-by-quarter narration scripts
- Scoring rubrics and evaluation criteria
- Investor demo guidance
- Troubleshooting tips
- Materials and resources list

**Key sections:**
- The Story You're Telling (narrative arc, your role as narrator)
- Quarter-by-Quarter Narration (detailed scripts for each quarter)
- Scoring the Journey (delivery, tech debt, API cost, collaboration)
- The Dice of Fate (optional chaos scenarios for Q3/Q4)
- The Final Reckoning (performance review and VP promotion)

### `UNIVERSAL_CHAOS_SCENARIOS.md`
**Six comprehensive crisis scenarios** (42KB) for optional use in Q3/Q4:

1. **Competitor Launched Superior App** (30-45 min) - Defensive pressure, rapid response
2. **Enterprise Customer Deal** (45-60 min) - B2B features for Fortune 500
3. **Regulatory Audit** (45-60 min) - Privacy and accessibility compliance
4. **Security Breach Discovered** (60 min) - Find and fix vulnerabilities
5. **Acquisition Due Diligence** (60-90 min) - Prove production-readiness
6. **Market Crash - Travel Downturn** (60 min) - Economic crisis, pivot to survive

Each scenario includes:
- Dramatic narrative setup to read aloud
- Department-specific challenges for all 8 roles
- Clear success criteria
- Gamerunner tips and variations
- Guidance on mixing scenarios across Q3/Q4

**Use chaos scenarios when:**
- Group is performing well and can handle pressure
- Energy needs a boost
- You want to simulate realistic startup chaos
- Time permits (scenarios add 30-90 min)

**Skip chaos when:**
- Group is struggling or behind schedule
- Directors need uninterrupted build time
- Already chaotic enough without events

## Quick Start

1. **Read GAMERUNNER.md** - Your complete facilitation guide
2. **Review chaos scenarios** - Decide if/when you'll use them
3. **Prepare materials** - Scoring sheets, d6 die (if using chaos), character sheets
4. **Set up room** - Ensure Claude access, screen sharing, timer
5. **Run the workshop** - Follow the quarter-by-quarter guide

## Tips

- **Timing is flexible**: Quarters can be 45-60 min based on group pace
- **Chaos is optional**: Use your judgment on whether to introduce scenarios
- **Role-play the NPCs**: Be the CEO, investors, customers - channel Silicon Valley energy
- **Celebrate wins**: Make demos feel impactful, recognize good work
- **Support struggling directors**: Offer guidance, reduce scope if needed
